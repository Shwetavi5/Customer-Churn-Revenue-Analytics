## Business Questions & Objectives

This project aims to analyze customer churn and revenue patterns using Python and SQL.  
The key business questions addressed are:

1. Which customers are most likely to churn?
2. How does contract type impact customer churn rates?
3. How does customer tenure influence churn probability?
4. Are higher-paying customers more likely to churn?
5. Which payment methods show the highest churn rates?
6. Which customer segments generate the highest revenue?
